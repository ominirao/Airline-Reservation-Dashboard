# Airline Reservation Dashboard

This project contains a Streamlit dashboard for airline booking analytics.

Files:
- app.py            : Streamlit app (root)
- airline_demo.db   : SQLite sample database
- sql/              : example SQL analysis scripts
- process_diagrams/ : placeholder diagrams
- documentation/    : BRD/FRD (text)

Run locally:
1. python -m venv .venv
2. source .venv/bin/activate
3. pip install -r requirements.txt
4. streamlit run app.py

Deploy:
- Push to GitHub and on Streamlit Cloud set App file = `app.py`.
